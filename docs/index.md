# Kick-Overlay Documentation Index

## Project Overview

**kick-overlay** is a full-stack streaming overlay system for Kick.com, featuring:
- Real-time chat overlays
- Drop game mini-game
- TTS (Text-to-Speech) integration
- Admin dashboard
- User profiles and points system

## Key Features

### Overlay System
- Configurable grid-based layout
- Widgets: Chat, Goals, Tips, Events
- Chroma key support for OBS
- Dynamic theme customization

### Drop Game
- Physics-based gameplay
- 6 powerups: TNT, Shield, Magnet, Ghost, Boost, PowerDrop
- Points and leaderboard
- Confetti/explosion effects

### TTS System
- 100+ ElevenLabs voices
- Per-user voice preferences
- Point cost for messages
- Audio queue management

### Points Economy
- Channel points (chat/watch)
- Drop points (gameplay)
- Powerup purchases
- Leaderboard rankings

## Known Issues for Refactoring

1. **Monolithic Server** - `index.ts` is 2300+ lines
2. **Large Components** - `AdminPage.tsx` is 1400+ lines
3. **Polling Architecture** - Could use WebSockets
4. **In-Memory State** - Lost on restart
5. **Duplicate Types** - Need centralization
